🚀 Quick Start

Get BoltDIY V2.0 running in minutes:

# Clone and install
git clone https://github.com/Stijnus/bolt.diy_V2.0.git
cd bolt.diy_V2.0
pnpm install
# Configure and start
cp .env.example .env.local
# Edit .env.local with your API keys
pnpm dev
